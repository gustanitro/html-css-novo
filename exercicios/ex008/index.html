Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)

Nessa frase temos um termo em destaque usando a tag <strong> (semântico)

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântica)

Nessa frase temos um termo em ênfase usando a tag <em> (semântico)

Texto Marcado

Nessa frase temos um termo marcado usando a tag <mark>

Texto Pequeno

Estamos criando um texto pequeno nesse parágrafo com a tag <small>

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado com a tag <dell>

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <ins>

Existe também o sublinhado com a tag <u>

Texto sobrescrito

Para inserir coisas do tipo x20+3 com a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O com a tag <sub>
